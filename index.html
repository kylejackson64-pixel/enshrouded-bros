⚒ Update 8 Changes Reference
⚔️
Combat Overhaul
  • Heavy attacks on all weapons
  • Weapon Ultimate system — fill meter for special move (wand AoE orb, 2H 360 cleave, dagger burst)
  • I-frames on dodge (unlock via "Strategic Maneuver" skill node)
  • New melee animations: gap closers, aerial strikes
  • Ledge-push on attack animations fixed
  • Enemy AI now dodges ranged attacks
  • Stealth system reworked — stealth damage improved
  • Aggro changes benefit stealth builds
🛡️
Gear & Itemisation
  • Armor perks return (HP, Mana, Stamina, Regen, Crit)
  • New perk: boost a skill tree node beyond its max level
  • Armor scales via upgrades — common and legendary start equal
  • Archetype focus: tanks get more armor, casters less
  • Damage recalculated — smaller numbers are NOT a nerf
  • Underperforming weapons and gems rebalanced
🌟
Skill Tree Rebuild
  • All skill points refunded on patch day
  • Full node repositioning, new skills added
  • Zoom-out view of entire tree added
  • Early nodes cheaper — reach builds faster
  • Double jump and key QoL unlocks easier to reach
  • Node levels: invest multiple points per node for deeper bonuses
🌐
Adventure Sharing + 1.0
  • Share your world/builds with the community
  • Gallery: browse other players' creations
  • V2 at 1.0: traps, puzzles, player-made challenges
  • Full 1.0 release: Autumn 2026 + console (PS5/Xbox)
  • Floating islands likely accessible at 1.0
